VPS

很早以前就一直想整理这么一个东西，整理一下VPS都能做什么

今天在尝试装一个软件的时候VPS出了一些问题，所以重新装了一些常用的软件，也顺便整理一下吧

Blog

写博客可以说是最基本的功能之一了

在github的gitpage中可以直接用的几个常用静态网页框架都可以直接在VPS上部署博客

复杂一些的还可以用Wordpress，像我就为了方便直接也是用了hexo

一般在重装VPS之后要部署博客都做这么几件事：

  1. 修改ssh端口
  2. 在root的目录下心间一个dir
  3. 将这个dir设置为bare的git repository
  4. 将博客clone在网站代理的目录

Anki Server

Anki是一个记忆的软件，可以规划好时间背单词，会自动按照记忆曲线分配每天需要背的任务，Anki也有非常强的功能，可以用CSS、html等内容定制模板

并且是一个全平台的软件，所以一个比较好的学习方式就是在电脑上学习时制作卡片，然后每天用手机来学习；

但是这个软件有个问题就是服务器在美国，而且由于人家是非盈利的，就导致国内速度特别慢；

而有一个VPS就可以在自己的VPS上部署服务器，这样同步速度完全就不是一个档次了

这还要专门装虚拟机就有点麻烦了，毕竟16位的程序一般也很少见了，这年头32位的程序都要逐渐被淘汰了，不过如果不能动态跑起来，至少看一下别人的WP达到能看懂的程度吧。

WP学习

首先还是根据错误字符串定位程序的关键代码入口

逆向的代码里看到有很多浮点运算相关的汇编指令，平常用的不是很多

大致流程看别人的write up梳理的很清楚

  1. 获取所有字符求和记为ldsum，长度记为len
  2. 计算cos(ldsum(sqrt(ldsum*d1)+ln(ldsum/2))+sin(len)*cos(len*ldsum))
  3. 将结果作为科学计数法保存为字符串，记录E前面的17位为一个长度17的字符串记为S17
  4. 通过17位中的8位进行一个简单的加减法，得到一个新的9位记为S9
  5. 上面9位的结果S9所有字符ASC求和+len和9*S17[4]+4*S17[5]+20比较是否相等

而对正常的每个字符，执行的操作是：

  1. var_7C = (Asc(Mid$(CStr(var_44), CLng(var_24), 1)) + Val(CStr(Left(var_44, 2))))
    1. CLng将var_24转换成长整形
    2. CStr将var_44转换成字符串
    3. Left函数返回var_44左数两个字符
    4. Val函数将字符串转换为数值，这个函数会在不能识别
    5. Mid函数将var_44左起第var_24位，长度1提取出来
    6. ASC函数返回字符串中第一个字符的ASCII码
  2. var_8C = Hex$(var_7c) 将其转换为Hex的值
  3. var_34 = 0 & Hex$(var_7C) 结果和0相与
  4. 最终结果要等于0817E747D7A7D7C7F82836D74747A7F7E7G7C7D826D817E7B7C
